semantica html5

cada etiqueta tiene un contexto, significado semántico, según donde se encuentre el flujo HTML

la función de las etiquetas, ahora, son especificar el motivo de cada elemento susceptible de identificación dentro de una web, para que los robots que la lean BOTS de los buscadores, sepan como clasificar esa información

  • posicionamiento en buscadores
  • controlar la búsqueda orgánica, de esta manera los bots posicionaran a tu pagina de manera consecuente con todo el contenido en Internet
  • compatibilidad con estándares web y navegadores futuros
