atributos data HTML5

se trata de unos atributos en etiquetas HTML, que te permiten almacenar datos que identifican al elemento en cuestión

debe tener como minimo un caracter de longitudud, no puede contener letras en mayusculas y debe llevar como prefijo: data-*

los atributos data-*y su contenido, seran totalmente ignorados por el userAgent


	<ul id="vegetable">
	  <li data-magnitud="10cm" data-caducidad="Marzo a Junio">Zanahoria</li>
	  <li data-magnitud="13cm" data-caducidad="Febrero a Marzo">Apio</li>
	  <li data-magnitud="3cm" data-caducidad="Marzo a Septiembre">Rábanos</li>
	</ul>

var data = "magnitud";
var liMagnitud = document.querySelectorAll("li[data-"+magnitud+"]"); //todos los <li> con data-magnitud
liMagnitud[0].getAttribut("magnitud"); //metodo antiguo
liMagnitud[0].dataset.magnitud; //'10cm'
liMagnitud[0].dataset.caducidad; //'Marzo a Junio'
$(liMagnitud[0]).data("magnitud"); //'10cm'

no se deberia usar...

  • Cuando tenemos etiquetas cuyo contenido semantico es mas apropiado para atribuir datos del elemento, como <time>, <mark>
  • no estan destinados para completar o susituir los micfroformatos, los atributos data no tendran uso público software externo, como los surfers, bots de buscadores, no interaccionaran con ellos,
