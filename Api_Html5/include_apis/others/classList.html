es una propiedad de todo elemento reconocido por el DOM, y por lo tanto por nuestras etiquetas html

element.classList

  • el.classList.remove("class")
  • el.classList.add("class" [, "class"])
  • el.classList.toggle("class")
    se puede añadir a un evento para que cadavez que se se dispare, intercambie la clase
  • el.classList.contains("class")
